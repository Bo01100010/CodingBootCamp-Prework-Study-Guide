HTML

  • head elements contain information about the Webpage, but cannot be seen by the viewer in a browser
  • body elements show the content of the webpage and are visable by the viewer in a browser
  • An Element is the name for the object that starts with angle bracket "<" and ends with ">"
  • A Tag is the name for additional infomation given to the the Element and is everything inside the angle brakets

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: Creates a pull request to Github to stage all changes
